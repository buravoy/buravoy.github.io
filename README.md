## Обзор проекта
Проект представляет собой веб-каталог вымышленной компании. 
Сайт включает адаптивный интерфейс, интерактивные модальные окна для заказа и чат с оператором.

## Основные технологии:
- HTML5 - структура страницы
- CSS3 (SCSS) - стилизация с использованием препроцессора
- JavaScript (Vanilla) - интерактивность и логика
- Адаптивный дизайн - поддержка мобильных устройств

## Структура проекта

```text
project/
├── index.html             # Основной HTML-файл
├── script.js              # JavaScript логика
├── style.scss             # SCSS стили
├── style.css              # Скомпилированные CSS стили
├── style.css.map          # Карта стилей
├── favicon.svg            # Иконка сайта
├── fonts/                 # Шрифты проекта
└── images/                # Изображения
```

### Брейкпоинты
- 1400px - уменьшение боковой панели
- 992px - мобильное меню
- 768px - адаптация для планшетов
- 576px - мобильные устройства
- 524px - мелкие экраны


## JavaScript Функционал


### Основные функции

####  Модальные окна
``` javascript
// Открытие модального окна заказа
// принимает на вход заголовок
function openModal(title)

// Закрытие модального окна
// при закрытие очищает данные окна 
function closeModal()
```

####  Валидация телефона
``` javascript
// Проверка правильности введенного номера
// принимает на вход ноду с элементами визуализации номера
function isPhoneCorrect(parent)
```

####  Чат
``` javascript
// Отправка сообщения в чат
function sendMessage()

// Генерация случайного ответа
function randomString(minWords = 5, maxWords = 10, minWordLength = 5, maxWordLength = 15)
```

####  Навигация
``` javascript
// Прокрутка к слайду
function goToSlide(id)
```




## Структура HTML
Основные разделы:
- Боковая панель (aside) - навигация по категориям
- Секция контента (section) - товары и информация
- Модальное окно заказа - ввод телефона
- Модальное окно чата - общение с оператором


## Настройка и развертывание
### Требования

- Современный браузер с поддержкой ES6+
- Веб-сервер для корректной работы шрифтов

### Шаги развертывания:
- Загрузить все файлы в корневую директорию
- Убедиться в наличии всех изображений и шрифтов
- При необходимости скомпилировать SCSS:

```bash
sass style.scss style.css
```
