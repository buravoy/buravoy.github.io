## Учебная практика 2025
Вариант 1:
# Frontend-разработка

**Тема недели:**
Разработка адаптивного лендинга с интерактивными элементами на чистом HTML/CSS/JS»

**Цель:** Закрепить навыки семантической верстки, адаптивного дизайна и базового программирования на JavaScript.

## Задание по дням:


- **День 1:** Проектирование. Выбрать тему для лендинга (портфолио фотографа, сайт для местного кафе, презентация книги). Создать прототип (скетч) на бумаге или в Figma. Подготовить контент (текст, подобрать изображения-заглушки).  
- **День 2:** Базовая верстка. Создать структуру HTML-документа с основными секциями (Header, Hero, About, Services/Features, Footer). Сделать семантическую разметку.  
- **День 3:** Стилизация. Написать CSS, используя Flexbox/Grid. Сверстать десктопную версию. Подключить шрифты.  
- **День 4:** Адаптивность и интерактив. Добавить медиа-запросы для планшетов и мобильных. Реализовать 2-3 интерактивных элемента на чистом JS (например: "бургер"-меню для мобильной версии, табы для переключения контента, модальное окно для формы, слайдер изображений на основе массива).  
- **День 5:** Оптимизация и публикация. Проверить код на валидаторе. Оптимизировать изображения. Выложить результат на GitHub Pages (просто и бесплатно). Задокументировать проект в README.  

**Итог:** Готовый, работающий, адаптивный сайт в открытом доступе. Минимум зависимостей, не требует настройки сервера.
